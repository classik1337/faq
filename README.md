# Hot key

Cntr+a
#### Copy
Cntr+c
#### Paste
Cntr+v
#### Save

#### ВЫРЕЗАТЬ!!!

# работа с git
### скачать проект
```bash
git clone <url>
```
### честь все файлы 
```bash
git add .
```
### сохрaнить комит
```bash
git commit -m "..."
```
### отправка на сервер
```bash
git push
```
